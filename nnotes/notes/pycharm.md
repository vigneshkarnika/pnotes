# pycharm

## Editor Basics
- option+enter -> intiutions
- cmd+shift+A -> help , show line numbers etc
- option+up -> selection
- comment cmd+/
- cmd+d duplicate
- cmd+del delete
- cmd+shift+up/down rearrange code
- cmd+,cmd- to expand shrink
- cmd+option+T - wrap
- ctrl+g - same line select
- cmd+ctrl+g= multiline select
- ctrl+space= suggestions
